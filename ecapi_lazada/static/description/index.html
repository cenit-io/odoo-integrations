How OMNA Integration Addon works?

Addon is powered by the OMNA Commerce API.
All connections, integration flows, and the data comes from this API which is available to your developer.
API developer can have full capability to integrate the information of orders, product, and order synchronization from original application.

OMNA Commerce API

OMNA Commerce API integrates with individual sales channel, allowing retailers to integrate multiple sales channel with just one API connection. Why spend the extra time and money on individual integrations when you can just connect to Commerce API?

OMNA Integration configuration

During installation you will have to provide the OMNA API url. Then you will be redirected to Cenit.io in order to complete the authentication process. If the process is completed successfully then you will be able to send or get data from OMNA in your behalf.

After installing the module, if you want to sign in again with OMNA. Go to the "Omna Integration/Settings/Sign in with Omna" menu and complete the authentication process.

Once the authentication process is completed, we proceed to import the integrations. To do this, go to the menu "Omna Integration / Settings / Import Products"

When the wizard opens, we will only click on the "Import Integrations" button and that's it.

To import products and their variants as well as the categories we will go to their respective wizards ... all located in "Omna Integration / Settings"

Once inside the wizard, you can choose the desired import method, providing you with exactly the desired import rules.

In addition, the app offers you a set of graphics that allow you to be aware of your company in real time.
