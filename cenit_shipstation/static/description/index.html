What you can do with this integration

You can send new shipments to Shipstation and update them, create warehouses, mark order as shipped, add fund and many others options.

Shipstation configuration

You will need to have an active ShipStation account. If you do not have one, you can easily request a free 30 day trial. Sign up with Shipstation

You will need Key, Secret and StoreId from Shipstation. You can find Key and Secret in Account Settings/Api Settings:




Now in Account Settings, go to Selling channels and copy the Id of the store.

Finally map your models

That's it! Every time you have a Sales Order in "sale" status, it will be sent automatically to Shipstation.

You can modify this behaviour and the fields sent to Shipstaion. Check the mapping under Mappings tab in Cenit module and create your own mappings.
