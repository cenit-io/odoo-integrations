Mailchimp configuration

During the installation of the module, you will have to provide your Mailchimp node service.

You can find it in your Maichimp account:

After the module is install you should authorize Cenit's app to send or get data from Mailchimp in your behalf. Go to your Cenit's account and select Authorizations menu.

In the Mailchimp authorization record, select Authorize action and authorize Cenit's app.


This is the default mapping of Odoo emails to Mailchimp campaigns:

You should set in the mapping a value for Mailchimp List and Template for sending campaigns. Select one List and one Template from your Mailchimp account, take theirs ids and copy them as values for both list_id and template_id in the mapping.

That's it! Every time an email is created, a campaign will be created automatically in Mailchimp.

You can modify this behaviour and the fields sent to Mailchimp. Check the mapping under Mappings tab in Cenit module and create your own mappings.
