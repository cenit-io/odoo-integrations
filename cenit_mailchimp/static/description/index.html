Setting up...

First things first: SignUp with MailChimp

From your Dashboard go to Accounts, and on the Extras tab select API Keys.

If you don't have an API Key you'll have to add one: write it down

Also take note of your Service Node

That's pretty much it

After installing the addon a form will let you set the data

Featuring...

Events:

Translators:

Out of the Box Flows so you can focus on what's really important.

  • Member Create

First steps

There's really only one thing you need to do: Map your models

  • Go to the Cenit IO Integrations menu.
  • Create a new Data type
    • Name it as you will.
    • Select the model you wish to send over to MailChimp (say 'Mass Mailing Contact').
    • Select the 'Mailchimp ' library
    • and the specific schema you want to use (say 'member.json ').
    • On the Mappings tab set how you want your mass mailing contacts to translate to Mailchimp
    • Save your work.

That's it! Everytime you create or modify a Mass mailing contact in Odoo, a member will be create on your Mailchimp account.

For more on mapping check the Documentation tab of the Cenit IO Integrations Client.
